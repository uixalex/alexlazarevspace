The Challenge+

Awaken180° had a proven in person weight loss methodology, but no digital tool to support clients between sessions. Tracking, motivation, and consistency all suffered as a result.

The Approach+

Mapped the full user journey to pinpoint the biggest friction points: onboarding confusion, logging drop off, and motivation loss after week two. Built the information architecture and feature priority around fixing those three moments.

Key Decisions+

Simplified onboarding to get users to their first action in under 2 minutes. Home screen surfaces only the 3 core daily actions, reducing decision fatigue and making consistency easier. Core actions like logging a meal or checking progress are completable in under 3 taps. Design system built for reuse, so new features can be added without breaking visual consistency.
